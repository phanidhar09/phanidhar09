## 🚀 Featured Project: ClozeHive

**[ClozeHive](https://github.com/phanidhar09/clozehive)** — a wardrobe intelligence platform with an AI stylist. 🔗 **Live: [clozehive.netlify.app](https://clozehive.netlify.app)**

A digital closet with outfit building, **FANI**-powered AI styling (Fashion AI Nurturing Individuality), travel & packing assistance, analytics, and **RAG**-enhanced insights grounded in your own items and wear history.

**Highlights**
- 🤖 **FANI AI Stylist** — LangGraph-style agent with MCP tool servers; structured chat, outfit recommendations, 5-tier outfit rating, daily style tips, and wardrobe gap detection
- 🔍 **RAG & vector search** — fashion knowledge retrieval, closet similarity, and purchase-gap analysis on **PostgreSQL + pgvector** embeddings
- 👁️ **Vision pipeline** — dedicated vision service for image analysis, smart ingest, and background removal
- ✈️ **Travel intelligence** — trips, AI-generated packing lists, and packing memory
- ⚡ **Real-time** — WebSocket notifications; JWT auth + Google OAuth with Redis-backed sessions

**Stack:** React + Vite + TypeScript · FastAPI microservices (API gateway, vision service, AI agent) · PostgreSQL + pgvector · Redis · nginx · Docker

```
Browser ──> nginx ──> React SPA
                └───> FastAPI Gateway ──> Vision Service (image analysis)
                            │       └───> AI Agent (FANI + MCP tools)
                            └───> Postgres/pgvector · Redis · GCS
```
